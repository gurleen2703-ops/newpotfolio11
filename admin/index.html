🚀 How to Deploy Changes to Live Site

1
Make your changes in Admin Panel
Edit content, add portfolio items, update testimonials — anything you want.
2
Click "⬇ Export sitedata.js"
This downloads a fresh sitedata.js file with all your changes baked in.
3
Replace the file in VS Code
Copy the downloaded sitedata.js into your project's data/ folder, replacing the old one.
4
Push to GitHub
Open VS Code terminal and run:
git add .
git commit -m "update site content"
git push
5
Vercel auto-deploys in ~20 seconds
Visit your Vercel dashboard — you'll see a new deployment triggered. Your live site updates automatically. ✅
💡 Tip: Bookmark your Vercel dashboard at vercel.com/dashboard to monitor deployments in real time.
